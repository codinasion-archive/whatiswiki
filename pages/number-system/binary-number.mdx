# Binary Number

A binary number is a number expressed in the **base-2** numeral system, which uses only two digits: **0** and **1**.
It is a fundamental concept in digital electronics and computer science, as all digital data and instructions
are represented using binary numbers.

## Representation

Each digit in a binary number is called a bit, and its value is determined by its position, or weight, within the number.

The rightmost bit has a weight of $2^0$ (1), the next bit to the left has a weight of $2^1$ (2), then $2^2$ (4), $2^3$ (8), and so on.

To convert a binary number to its decimal equivalent, the value of each bit is multiplied by its weight and the results are summed.

For example, the binary number **1011** represents the [decimal number][decimal_number_what_is_wiki] (1 x $2^3$) + (0 x $2^2$) + (1 x $2^1$) + (1 x $2^0$),
which is equal to **11**.

## Uses

Binary numbers are used extensively in digital electronics and computer science for a variety of tasks, including:

- **Representing data**: All digital data, such as text, images, and sound, is represented using binary numbers.
- **Performing arithmetic**: Binary arithmetic is used in computer processors and memory to perform mathematical operations.
- **Storing and retrieving information**: Binary numbers are used to represent addresses and data in computer memory.
- **Performing logical operations**: Binary numbers are used in boolean algebra to represent logical values (true or false) and perform logical operations (and, or, not, xor).
